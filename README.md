# 以下デモアプリの内容

## サービス概要
同窓会や飲み会などの多人数の予定を組む際に、他の参加者が直接入力するアプリだと<br>
日本人特有の表現でおおむね行けるが万が一があると難しいということで△などの曖昧な表現で<br>
幹事が直接聞きに行くという手間が発生してしまいます。<br>

そのためこのアプリでは幹事が直接参加者に予定を聞くことを前提として<br>
直接聞いた内容を管理しやすくすることを目的とし、幹事の思考時間の短縮や、<br>
幹事の予定調整、計画のサポートを目指します。

## 開発背景
以前友人を集め10人程度の小さい同窓会を開こうとした際に<br>
既存のアプリを使用しましたが△を多用されてしまい、直接聞き<br>
Excelを使用して管理するという結果となってしまいました。<br>
日本人特有のニュアンスも含めて予定を調整するのであれば<br>
直接聞いたうえで幹事が管理したほうが良いと思い、<br>
幹事が直接入力することを前提としたアプリを作成したいと考えました。

## 想定ユーザー
同窓会や飲み会などの幹事<br>
部活動のリーダー<br>
小規模グループのリーダー<br>

## 利用イメージ
アプリを開く<br>
表の参加者欄に参加者の名前を入力する。<br>
日にちと参加者の予定ごとに〇か×を押す<br>
表示されている参加不参加の集計数で候補日を決める。<br>
カレンダーで確認を行う。

## 機能一覧

| 機能名 | 内容 |
|--------|------|
| 人物名入力機能 | 名前を入力し表に反映させる機能 |
| 予定入力機能 | 予定を〇と×で入力し可視化する機能 |
| 人数集計機能 | 日付ごとの〇と×の人数を集計する機能 |
| カレンダー表示機能 | 集計された人数をカレンダーで表示する機能 |
| 予定ごとの記録機能 | 予定ごとにタイトルを付けて保存できる機能 |

## 完成後追加したい機能

写真を保存しておく機能
